
1. Abra o prompt de comando (CMD.EXE)
2. Crie um novo diretório chamado "MeusDocumentos" dentro do seu diretório HOME usando o comando `mkdir`.
3. Altere o diretório atual para o diretório "MeusDocumentos" e, em seguida, exclua-a usando o comando `rmdir`.
4. Crie um novo diretório chamado "Downloads" dentro da pasta "MeusDocumentos" usando o comando `mkdir`.
5. Altere o diretório atual para o diretório "Downloads" e, em seguida, crie uma nova pasta chamada "Vídeos" usando o comando `mkdir`.
6. Altere o diretório atual para o diretório "MeusDocumentos" e, em seguida, crie um novo arquivo chamado "MeuArquivo.txt" usando o comando `echo oi > MeuArquivo.txt`.
7. Exclua o arquivo do exercício 6 o usando o comando `del`.
8. Altere o diretório atual para o diretório "Downloads" que você criou 4. e, em seguida, crie um novo arquivo chamado "NovoArquivo.txt" usando o comando `echo`, como no exercício 6.
9. Exclua "NovoArquivo.txt".
